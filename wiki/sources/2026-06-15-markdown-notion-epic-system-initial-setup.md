# Notion Epic - 시스템 초기 설계 및 구축

## Summary

Laimory 개발을 위한 초기 시스템 아키텍처, 개발 환경, 운영 환경 구축 epic.

## Key Claims

- 목표는 App, Server, AI 서비스가 안정적으로 개발될 수 있는 초기 구조를 잡는 것이다.
- 전체 범위에는 서비스 아키텍처, App/Server/AI 통신 구조, 개발/운영 환경, GitHub 협업 룰, 공통 개발 프로세스, CI/CD 및 배포 전략이 포함된다.
- App 범위에는 모바일 앱 초기 구성, 앱 아키텍처/상태 관리, 디자인 시스템, 로그인/기본 화면, 네트워크 환경, 앱 배포/테스트 환경이 포함된다.
- Server 범위에는 서버 초기 구성, DB, 인증/사용자 관리, API 설계, 클라우드 인프라/배포, 모니터링/로깅, AI 서버 연동 기반이 포함된다.
- AI 섹션은 캡처 시점에서 비어 있다.

## Caveats

- Epic은 초기 계획이며, 구현 완료 상태를 증명하지 않는다.
